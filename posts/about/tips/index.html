From what age can kids learn to ski?

From what age can kids learn to ski? Children can start on skis from about 2 years old, at this age it simply starts with learning to balance on absolutely flat snow on a pair of skis, at this age it is a very slow process, so be prepared with patience! The 2-3 year old child will get bored or exhausted with the activity quite quickly so it best to start out slow, if the child gets enough, it’s better to stop for the day or have a break as it’s meant to be fun not a chore. It’s never pleasant to see crying children not having fun, as it’s meant to be fun!
